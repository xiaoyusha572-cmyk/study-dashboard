# 一颗沙莉

个人学习推进与每日复盘仪表盘。第一版为纯前端静态网站，不需要安装依赖。

## 功能

- 按 DDL 展示大学物理、高等数学、电路理论和英语四级任务
- 自动选出三项今日推进任务
- 每日打开时选择动态调色盘
- 随时增删改总体任务、子任务和 DDL
- 手动替换每日任务，或按 DDL 自动均匀分配
- 在月历中查看接下来每天的未完成安排
- 使用 12 周热力图查看每日积累
- 自动生成每周复盘卡片和最近 DDL 倒计时
- 完成任务后随机获得温柔反馈和休息建议
- 按已完成科目解锁 F1 风格阶段里程碑
- 导出和导入 JSON 备份，方便换设备前手动迁移
- 本地保存任务完成状态
- 完成任务时显示进度奖励动画
- 支持每日复盘记录
- 适配手机端

## 本地预览

直接打开 `index.html`，或在项目目录运行：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## GitHub Pages

将仓库上传至 GitHub 后，在仓库的 `Settings > Pages` 中选择从默认分支根目录部署。

当前版本使用浏览器本地存储：同一台设备上的修改会立即保存。若需要手机和电脑之间实时同步，可在下一阶段接入 Supabase 云数据库。
